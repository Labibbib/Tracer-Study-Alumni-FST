# Tracer Study Alumni Fakultas Sains dan Teknologi UIN Jakarta

## Deskripsi
Tracer study merupakan bagian penting dari proses pelaksanaan akademik sebagai umpan balik sehingga kualitas pendidikan di Fakultas khususnya program sarjana dan pascasarjana dapat terjaga dan semakin lebih baik untuk di masa mendatang. Data tracer study ini sangat membantu dalam proses akreditasi, penyusunan kurikulum dan menguatkan jejaring alumni. Tracer Study ini menggunakan data alumni Fakultas Sains dan Teknologi (FST) yang dikumpulkan melalui gform dalam rentang waktu tertentu. Sebagai Data Analis, tugas kami kali ini adalah menganalisa tracer study alumni FST dan memberikan insight sekaligus rekomendasi kepada Rektor dan ketua Puskarir UIN Jakarta.
